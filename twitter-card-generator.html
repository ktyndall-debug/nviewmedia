How to Save Your Twitter Card Image:

  1. Take a screenshot of the card below (on Mac: Cmd + Shift + 4, on Windows: Windows + Shift + S)
  2. Save the image as twitter-card.jpg in your assets folder
  3. The image dimensions should be 1200x630 pixels (already set below)
  4. Make sure the image is under 5MB in size
  5. Upload the image to your web server in the /assets/ directory
  6. The meta tags in your HTML are already configured to use this image
  7. Test your Twitter card using the Twitter Card Validator
