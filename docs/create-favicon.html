NView Media Favicon Creator

Instructions to Create favicon.ico:

  1. Click each download button to save the PNG files
  2. Go to favicon.io/favicon-converter/ or icoconvert.com
  3. Upload the PNG files you downloaded
  4. Download the generated favicon.ico file
  5. Place it in the root directory of your website

Alternative: Using ImageMagick (if installed):

convert favicon-16.png favicon-32.png favicon-64.png favicon.ico
